# การพัฒนาชุดเครื่องมือการเรียนรู้เชิงปฏิบัติการสำหรับการใช้งาน Post-Quantum Cryptography (Practical PQC Learning Kit)

ท่ามกลางการเปลี่ยนแปลงครั้งสำคัญที่เทคโนโลยีควอนตัมกำลังกระทบต่อระบบความมั่นคงปลอดภัยไซเบอร์ทั่วโลก
การเตรียมพร้อมของประเทศจำเป็นต้องเริ่มต้นจากการสร้าง "ฐานความรู้" (Knowledge Base) และ
"ทรัพยากรมนุษย์" (Human Resource) ที่สามารถเข้าใจ ปรับตัว และประยุกต์ใช้แนวทาง Post-Quantum
Cryptography (PQC) ได้อย่างต่อเนื่องและยืดหยุ่น

โครงการนี้มีวัตถุประสงค์เพื่อพัฒนา “ชุดเครื่องมือการเรียนรู้ Post-Quantum Cryptography” หรือ PQC
Learning Kit ซึ่งเป็นแหล่งความรู้ open source ที่เปิดให้คนไทยสามารถเรียนรู้ ฝึกปฏิบัติ
และแบ่งปันเนื้อหาด้าน PQC ได้อย่างอิสระ โดยเน้นความเข้าใจเชิงประยุกต์และสามารถนำไปใช้กับระบบจริง
ไม่ว่าจะเป็นระบบ PKI, TLS, การเซ็นเอกสารดิจิทัล, และโครงสร้างพื้นฐานไซเบอร์ของประเทศ
เหมาะสมแก่การใช้ในสถาบันการศึกษา หน่วยงานราชการ และการอบรมในระดับต่างๆ

ชุดเครื่องมือนี้จะทำหน้าที่เป็นก้าวแรกในการยกระดับศักยภาพของประเทศในการเปลี่ยนผ่านเข้าสู่ยุคความมั่นคงปลอดภัยแบบ
Quantum-Safe โดยเน้นการสร้างบุคคลที่มีความรู้สามารถรับมือความเปลี่ยนแปลง
และออกแบบทางออกที่เหมาะสมกับบริบทในอนาคตอันไม่แน่นอนได้

## วัตถุประสงค์

- สร้างฐานความรู้แบบเปิด (open knowledge base) ด้าน Post-Quantum Cryptography (PQC) ที่คนไทยสามารถเข้าถึง เรียนรู้ ฝึกฝน และแบ่งปันได้อย่างอิสระ
- ส่งเสริมการพัฒนาทรัพยากรมนุษย์ที่มีทักษะและความเข้าใจในการนำ PQC ไปประยุกต์ใช้กับระบบจริง ทั้งในระดับแอปพลิเคชันและโครงสร้างพื้นฐาน
- สนับสนุนการเปลี่ยนผ่านระบบดิจิทัลของประเทศไปสู่ความมั่นคงปลอดภัยแบบ Quantum-Safe ผ่านการเรียนรู้ที่เน้นการปฏิบัติ (hands-on) และเหมาะสมกับบริบทไทย
- วางรากฐานเพื่อชุมชนโอเพนซอร์สสำหรับ PQC ที่ทุกภาคส่วนสามารถมีส่วนร่วมในการพัฒนา สอบทาน และเผยแพร่เนื้อหา เพื่อให้ทันต่อวิวัฒนาการของภัยคุกคามจากเทคโนโลยีควอนตัม
- เพิ่มความยืดหยุ่นในการปรับใช้มาตรฐาน PQC ที่ยังคงพัฒนาอย่างต่อเนื่อง ด้วยการฝึกฝนทักษะการออกแบบ ปรับใช้ และย้ายระบบเดิมให้รองรับเทคโนโลยีใหม่ได้อย่างมั่นใจ

## กลุ่มเป้าหมาย

- นักเรียน/นักศึกษามหาวิทยาลัยระดับปริญญาตรีขึ้นไปในสาขาวิทยาการคอมพิวเตอร์ วิศวกรรมคอมพิวเตอร์ และ cybersecurity
- วิศวกรและนักพัฒนาระบบความปลอดภัย
- หน่วยงานภาครัฐหรือเอกชนที่เกี่ยวข้องกับการจัดการข้อมูลสำคัญหรือบริการที่ต้องการความปลอดภัยระดับสูง
- สถาบันอบรมและเจ้าหน้าที่ด้านนโยบายในระดับประเทศ

## ขอบเขตเนื้อหา

หมวดที่ 1: ความรู้พื้นฐาน

- ความรู้เบื้องต้นเกี่ยวกับ Quantum Computing
- ผลกระทบของ Quantum Computing ต่อ Cryptography
- มาตรฐานที่เกี่ยวข้องกับ PQC
  - สถานะของการคัดเลือก PQC algorithms โดย NIST
  - ลักษณะของ algorithm ที่ผ่านการคัดเลือก และกรณีใช้งานที่เหมาะสม

หมวดที่ 2: ตัวอย่างการใช้ PQC ในรูปแบบต่างๆ

- การติดตั้งระบบ PKI ที่รองรับ PQC
- การสร้าง PQC key pair 
- การเซ็นเอกสารแบบดิจิทัลด้วย PQC
- การติดตั้ง PQC TLS certificate บน reverse proxy
- การใช้ PQC ทำ End-to-end encryption

หมวดที่ 3: การปรับใช้งานจริงในระบบที่มีอยู่

- ความไม่แน่นอนของ algorithm ใหม่ซึ่งอาจจะถูกถอดจากมาตรฐานในภายหลัง
- การสำรวจทำบัญชี cryptographic algorithm ที่กำลังถูกใช้
- การใช้ cryptography แบบผสมระหว่าง Post-Quantum และ Classical
- ความพร้อมของโปรแกรมฝั่งผู้ใช้งานทั่วไป
- การรับมือกับขนาด key และเวลาประมวลผลที่เพิ่มขึ้น

## แผนการดำเนินงาน

- ใช้บุคลากรภายใน NECTEC ในการรวบรวมข้อมูล, วางโครงสร้างเนื้อหา, สร้างต้นแบบเครื่องมือ
  และสร้างกิจกรรมการเรียนรู้เชิงปฏิบัติ
- ปรึกษาผู้เชี่ยวชาญภายนอก เพื่อทบทวนเนื้อหา ตรวจสอบความถูกต้องเชิงเทคนิค
  และให้ข้อเสนอแนะด้านความเหมาะสมของแนวทางและเครื่องมือ
- ปรับปรุงเนื้อหาและชุดเครื่องมือให้ดียิ่งขึ้น ตามความคิดเห็นและข้อเสนอแนะจากผู้เชี่ยวชาญ
- จัดกิจกรรม workshop เพื่อนำเสนอและทดลองใช้ PQC Learning Kit ร่วมกับกลุ่มเป้าหมายจริง
  เพื่อรวบรวมข้อเสนอแนะเชิงปฏิบัติ รวมถึงส่งเสริมการนำไปทดลองใช้งานและการมีส่วนร่วมจากชุมชน
- สรุปผลการดำเนินงาน วิเคราะห์ข้อเสนอแนะ และติดตามการใช้งานจริงจากบุคคลภายนอก

### การจ้างที่ปรึกษาผู้เชี่ยวชาญ

- เพื่อทบทวนเนื้อหาทางเทคนิคว่ามีความถูกต้องตรงตามหลักวิชาการ และมีความทันสมัยตามสถานการณ์ปัจจุบัน
- เพื่อประเมินความเหมาะสมเบื้องต้นของชุดเครื่องมือในการประกอบการสอน
- เพื่อสร้างความน่าเชื่อถือ

  การมีผู้เชี่ยวชาญภายนอกที่มีประสบการณ์หรือมีบทบาทในองค์กรวิชาชีพ เช่น สถาบันการศึกษา
  หน่วยงานมาตรฐาน หรือองค์กรความมั่นคง จะช่วยเพิ่มความน่าเชื่อถือให้กับ PQC Learning Kit
  และเอื้อต่อการผลักดันให้หน่วยงานอื่นนำไปใช้งานหรือนำเข้าไปในหลักสูตรทางวิชาการหรืออบรมอย่างเป็นทางการ

### การจัด Workshop

- รวบรวมข้อเสนอแนะจากผู้ใช้จริง (Collect Feedback)

  การจัด Workshop เปิดโอกาสให้ผู้เข้าร่วมที่เป็นกลุ่มเป้าหมายหลัก เช่น นักศึกษา นักพัฒนา บุคลากรด้าน
  IT ได้ใช้ชุดเครื่องมือในบริบทจริง เป็นช่องทางที่มีประสิทธิภาพในการเก็บรวบรวมข้อเสนอแนะ ปัญหา
  จุดสับสน หรือข้อเสนอการปรับปรุง ซึ่งจะถูกนำไปใช้ในการปรับปรุงเนื้อหา ความเหมาะสม
  ของชุดเครื่องมือต่อไป

- ประเมินประสิทธิภาพของชุดการเรียนรู้ (Evaluate Effectiveness)

  Workshop สามารถใช้เป็นเวทีในการประเมินผลทั้งในเชิงปริมาณและเชิงคุณภาพ
  โดยมีการวัดผลผู้เรียนก่อนและหลังการใช้งาน PQC Learning Kit รวมถึงประเมินความเข้าใจ
  ความสามารถในการใช้งาน และทักษะที่ได้รับ เพื่อวิเคราะห์จุดแข็งและจุดที่ควรปรับปรุงของชุดเรียนรู้

- สร้างชุมชนผู้ใช้งานและผู้พัฒนา (Build Community)

  การเรียนรู้ร่วมกันผ่าน Workshop จะเป็นจุดเริ่มต้นของการสร้าง "ชุมชน" (Community)
  รอบชุดเครื่องมือแบบโอเพนซอร์ส ซึ่งเป็นสิ่งสำคัญในการต่อยอดโครงการในอนาคต
  ไม่ว่าจะเป็นการร่วมพัฒนาเนื้อหา การแจ้งปัญหา หรือการมีส่วนร่วมในการเผยแพร่ไปสู่องค์กรอื่นๆ

## งบประมาณ

งบประมาณแบ่งเป็นสามส่วนหลัก คือ

- ค่าจ้างที่ปรึกษาภายนอก
- ค่าจัดการ workshop
- ค่าดำเนินการ NECTEC

## ผลงานส่งมอบ

- บทเรียน, ชุดเครื่องมือ, ตัวอย่าง, และกิจกรรมเชิงปฏิบัติ
- ผลการประเมินจากการจัด workshop
